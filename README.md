Deaf Talk 

Deaf Talk is an assistive communication system designed to bridge the gap between hearing-impaired and speech-impaired individuals.

This project converts speech into text and also translates input into sign language representation, enabling inclusive and accessible communication.

Project Overview

Communication barriers between hearing and speech-impaired individuals can limit daily interactions.

The goal of this system is to:

Convert spoken words into real-time text

Display corresponding sign language representation

Improve accessibility using modern web technologies

Key Features

🎤 Real-Time Speech Recognition (Speech → Text)

🔤 Instant Text Display

🤟 Text to Sign Language Representation

⚡ Fast and Responsive Interface

🌐 Web-Based Application

📱 User-Friendly Design

How It Works

User clicks the microphone button

System captures speech input

Speech is converted into text using Speech Recognition API

Text is displayed on the screen

Corresponding sign language representation is shown
